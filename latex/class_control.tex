The \mbox{\hyperlink{class_control}{Control}} class to control the program logic. 

The \mbox{\hyperlink{class_control}{Control}} class will control the the program logic and manage the communication between the model and view objects it will also contain some variable about the current state of the program like the curently selected folder path.
